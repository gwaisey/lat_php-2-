# 🚀 PHP Basic Logic & Algorithm Exercises

This repository contains a collection of PHP-based solutions for various programming logic challenges. These exercises are designed to strengthen core problem-solving skills, focusing on string manipulation, mathematical sequences, and algorithmic patterns.

## 📌 Project Overview
The solutions in this repository address common fundamental programming tasks:

- **String Manipulation**: Counting characters, word splitting, and custom formatting.
- **Data Validation**: Date validation logic (handling leap years/days) and input verification.
- **Mathematical Sequences**: Implementation of Fibonacci series and square number patterns.
- **Visual Logic (Asterisk)**: Generating various geometric shapes using nested loops.
- **Advanced Systems**: 4-digit alphanumeric counters (e.g., 0009 to 000A) and number-to-word conversion.

## 🛠️ Tech Stack
- **PHP 8.x** (Core Logic)
- **XAMPP / Laragon** (Local Environment)
- **Git** (Version Control)
